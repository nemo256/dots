# Firefox
- sign in
- set network.protocol-handler.expose.magnet -> false (magnet links)
- set layers.acceleration.force-enabled -> true (hardware acceleration)
- install plugins:
  * AdBlocker for YouTube
  * hide-scrollbars
  * Privacy Badger
  * uBlock Origin
  * Vimium
  * Firefox Color

- install themes:
  * IndigoDark
